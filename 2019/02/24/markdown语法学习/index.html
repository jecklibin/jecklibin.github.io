markdown语法学习

Markdown
  • Markdown

Markdown是一种纯文本格式的标记语言，它语法和操作都很简单，让写作者摆脱排版的烦恼，而且很多平台都是支持Markdown编辑模式的。笔者因为使用hexo，它的文章都是Markdown格式的，所以第一要务就是学习Markdown以方便后续的写作。

标题

文字前面加＃表示标题，＃表示一级标题，＃＃表示二级标题，依此类推，支持六级标题。

示例：

效果如下：

这是一级标题

这是二级标题

这是三级标题

这是四级标题

这是五级标题
这是六级标题
